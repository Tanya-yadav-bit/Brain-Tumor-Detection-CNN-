<h2>📌 Project Overview</h2>
<p>
This project focuses on detecting brain tumors from MRI images using a <b>Convolutional Neural Network (CNN)</b>. 
It includes data preprocessing, model training, evaluation, and advanced visualization techniques like 
<b>Grad-CAM</b> to interpret model predictions.
</p>

<p><b>This project demonstrates deep learning, image classification, and model interpretability skills.</b></p>

<hr>

<h2>🎯 Objectives</h2>
<ul>
  <li>Classify MRI images into <b>tumor</b> and <b>non-tumor</b></li>
  <li>Apply <b>CNN architecture</b> for image classification</li>
  <li>Improve model performance using <b>Dropout and pooling</b></li>
  <li>Visualize model behavior using <b>Grad-CAM</b></li>
</ul>

<hr>

<h2>🛠️ Tools & Technologies</h2>
<ul>
  <li>Python</li>
  <li>TensorFlow / Keras</li>
  <li>NumPy, Matplotlib</li>
  <li>OpenCV</li>
</ul>

<hr>

<h2>⚙️ Model Architecture</h2>
<ul>
  <li>Multiple <b>Conv2D + ReLU</b> layers</li>
  <li><b>MaxPooling</b> for feature reduction</li>
  <li><b>Dropout</b> for overfitting prevention</li>
  <li>Fully connected <b>Dense layer</b></li>
  <li><b>Sigmoid activation</b> for binary classification</li>
</ul>
